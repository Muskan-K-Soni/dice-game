# Dice Game 🎲

A fun and interactive dice game built using HTML, CSS, and JavaScript.  
Players can roll the dice using the Roll Dice button, watch the dice shake with animation, and see the winner along with a live score update.

---

## Features

- Roll dice with a single click
- Random dice number generation
- Dice shake animation before showing result
- Displays winner between Player 1 and Player 2
- Live scoreboard updates after every round
- Responsive and simple UI

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (DOM Manipulation, Events, Logic)

---

## What I Learned

Through this project, I learned:

- How to generate random integer numbers using `Math.random()` and `Math.floor()`
- DOM element selection and manipulation
- Changing image sources dynamically with `setAttribute()`
- Updating webpage text dynamically using `innerHTML`
- Using `if-else` statements for game logic
- Creating click events with `addEventListener()`
- Applying CSS animations
- Using `setTimeout()` for delayed actions
- Managing score using JavaScript variables

---

## How to Run the Project

1. Download or clone the repository
2. Open `index.html` in your browser
3. Click the **🎲 Roll Dice** button to play
